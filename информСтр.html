МІЙ ЕЛЕКТРОННІЙ ДОВІДНИК HTML
Призначення Назва тегу Короткий опис Стандарт
Основні теги <Html> Вказує програмі перегляду сторінок, що це HTML документ. HTML5
<head> Визначає місце, де поміщається різна інформацію не відображається в тілі документа. Тут розташовується тег назви документа та теги для пошукових машин HTML5
<! DOCTYPE> Визначає тип документа. HTML5
<body> Визначає видиму частину документа HTML5
Теги змісту <title> Поміщає назва документа в зміст програми перегляду сторінок HTML5
Теги форматування тексту <pre> Обромляє попредньо відформатований текст HTML5
<h1> Створює найбільший заголовок(як окремий абзац) HTML5
<h6> Створює найменший заголовок HTML5
<b> Створює жирний текст HTML5
<i> Створює похилий текст HTML5
Форматування <p> Створює новий параграф HTML5
<P align =?> Вирівнює параграф щодо однієї зі сторін документа, значення: left, right, justify або center HTML5
<ol> Створює нумерований список HTML5
<li> Визначає кожен елемент списку і присвоює номер HTML5
<ul> Створює ненумерований список HTML5
<Div align =?> Тег, який використовується для форматування великих блоків тексту HTML документа. Часто використовується в таблицях стилів HTML5
Графічні елементи <Img src = "name"> Додає зображення в HTML документ HTML5
<Img src = "name" align =?> Вирівнює зображення до однієї зі сторін документа, приймає значення: left, right, center; bottom, top, middle HTML5
<Img src = "name" border =?> Встановлює товщину рамки навколо зображення HTML5
<Hr> Додає в HTML документ горизонтальну лінію. HTML5
<Hr width=?> Встановлює ширину лінії, можна вказати ширину в пікселях або відсотках HTML5
Таблиці <tr> Визначає рядок в таблиці HTML5
<table> Створює таблицю HTML5
<td> Визначає окрему клітинку в таблиці HTML5
<th> Визначає заголовок таблиці (нормальна осередок з відцентрованим жирним текстом) HTML5
Атрибути таблиці <Table border = #> Задає товщину рамки таблиці HTML5
<Table width = #> Встановлює ширину таблиці в пікселах або відсотках від ширини документа HTML5
<Tr align =?> Або align =?> Встановлює вирівнювання в клітинках таблиці, приймає значення: left, center, або right . HTML5
<Td colspan = #> Вказує кількість стовпців яку об'єднано в одній клітинці. (За замовчуванням = 1) HTML5
